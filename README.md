# E-commerce Brazil - End-to-End Analytics Report 🛍️

Final project for the **Data Analyst Program at Cesae Digital (Porto)**. This comprehensive Power BI report transforms a complex e-commerce dataset into a strategic decision-making tool.

### 📈 Key Insights & Business Value
- **Logistics:** Identified a 12-day average delivery time, suggesting regional distribution centers to reduce costs and lead time.
- **Inventory Management:** Applied **ABC Classification** to prioritize top-selling categories (80% of revenue).
- **Customer Intelligence:** Segmented 96k+ customers using **RFM Clustering**, distinguishing loyal high-spenders from at-risk inactive users.

### 🛠️ Technical Implementation
- **Data Modeling:** Robust Star Schema with 10+ connected tables.
- **Advanced DAX:** Measures for Year-to-Date (YTD) comparisons, Pareto accumulation, and average delivery time.
- **UI/UX:** Navigation menu, clean dark-themed dashboard, and interactive tooltips.
- **Geography:** Dynamic map analysis of sales concentration in Brazil (São Paulo/Rio focus).

*This report includes actionable suggestions for Logistics, Stock Management, and Customer Retention.*
